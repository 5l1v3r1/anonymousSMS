# anonymousSMS
#### send free anonymous sms

## Installing in termux

* `apt update`
* `apt upgrade`
* `apt install git`
* `git clone https://github.com/mishakorzik/anonymousSMS`
* `cd anonymousSMS`
* `bash setup.sh`

#### Succesfull installed in termux!

## Installing in linux

* `apt update`
* `apt install git`
* `git clone https://github.com/mishakorzik/anonymousSMS`
* `cd anonymousSMS`
* `bash setup.sh`

#### Succesfull installed in linux!
